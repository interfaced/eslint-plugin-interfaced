# interfaced/lines-between-methods

Enforce newlines between methods.

**Fixable:** this rule is automatically fixable by `--fix`.

## Options

```
{
	amount: <number>
}
```

#### amount

How many newlines should be between methods (including JSDoc). The default value is `1`.
