# interfaced/lines-between-props

Enforce newlines between properties.

**Fixable:** this rule is automatically fixable by `--fix`.

## Options

```
{
	amount: <number>
}
```

#### amount

How many newlines should be between properties (including JSDoc). The default value is `1`.
